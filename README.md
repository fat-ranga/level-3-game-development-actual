All scripts are in the source folder.

Link to testing video: https://www.youtube.com/watch?v=xifR2EySnVQ

# Controls
Tab - open inventory.\
WASD - movement.\
Esc - to go back in menus.\
F11 - toggle fullscreen.\
Left-click - punch.\
Right-click - zoom.\
F - interact.\
Space - jump.
